Designed and built a large-scale holiday lighting display featuring more than **7,000 individually addressable RGB LEDs** synchronized to music.

**Highlights**

* 7,000+ programmable RGB pixels
* Music synchronization
* Custom animation sequencing
* Large-scale controller networks
* Power distribution design
* Seasonal deployment and maintenance
* Outdoor system troubleshooting

**Technology**

* xLights
* E1.31 / sACN Networking
* Raspberry Pi
* Linux
* Addressable RGB Pixels
